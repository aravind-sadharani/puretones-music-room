## PureTones Analysis Tools

At PureTones, we aim to improve the understanding of scales and tuning systems in Indian Classical music. To this end, we provide the following tools for building and analyzing Indian Musical scales and for analyzing the consonance between Drone and Scale tunings.

* [Scale Builder](/scalebuilder/) is a tool to build microtonally accurate scales for Ragas in Indian Classical music.
* [Drone Analyzer](/droneanalyzer/) is a tool to analyze the tuning system induced by a drone tuning and to visualize how it matches up with respect to a scale tuning.
