## Scale for Raga Shankarabharanam

The scale for Raga Shankarabharanam can be built using the [Scale Builder](/scalebuilder/) tool with the following inter-note relationships.

```

(Sa,ma) = I(m)
(Sa,Pa) = I(P)
(ma,Dha) = I(G)
(Re,Dha) = I(P)
(Ni',Re) = (Ga,Pa)
```
