## Common Pitch

## Scale for Raga Kalyani

The scale for Raga Kalyani can be built using the [Scale Builder](/scalebuilder/) tool with the following inter-note relationships.

```
(Sa,Pa) = I(P)
(Re,Pa) = I(m)
(Re,Ma) = I(G)
(Ga,Pa) = (Ma,Dha)
(Ma,Ni) = I(m)
(Ga,Dha) = I(m)
```
