## Scale for Raga Bhairav

The scale for Raga Bhairav can be built using the [Scale Builder](/scalebuilder/) tool with the following inter-note relationships.

```
(Sa,Pa) = I(P)
(Sa,ma) = I(m)
(Sa,Ga) = I(G)
(Ga,dha) = I(G)
(re,dha) = I(P)
(Ni,Sa") = (Sa,re)
```
